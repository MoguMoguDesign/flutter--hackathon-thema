---

Use the Task tool with subagent_type='flutter-component-specialist' to help with Flutter UI component development.

The flutter-component-specialist agent specializes in:
- Creating reusable UI components
- Implementing Material Design and custom design systems
- Building responsive layouts
- Writing widget tests
- Performance optimization
- Accessibility implementation

Example usage:
- "新しいカスタムボタンコンポーネントを作成して"
- "既存のカードウィジェットをリファクタリングしたい"
- "レスポンシブな入力フィールドを実装して"
