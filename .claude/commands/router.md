Use the Task tool with subagent_type='go-router-navigator' to help with go_router navigation implementation.

The go-router-navigator agent specializes in:
- Setting up GoRouter configuration
- Implementing type-safe routing with GoRouteData
- Creating navigation guards and redirects
- ShellRoute and StatefulShellRoute patterns
- Deep linking support
- Custom page transitions

Example usage:
- "アプリのルーティング設定をセットアップして"
- "認証ガードを実装したい"
- "型安全なルーティングに移行して"
- "タブナビゲーション用のShellRouteを作成して"
