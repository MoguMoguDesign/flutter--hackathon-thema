---
description: Launch github-issue-creator agent to create GitHub issues
---

Use the Task tool with subagent_type='github-issue-creator' to help with creating GitHub issues.

The github-issue-creator agent specializes in:
- Creating bug reports
- Writing feature requests
- Using issue templates
- Providing clear reproduction steps
- Including relevant context

Example usage:
- "バグレポートのイシューを作成して"
- "新機能のリクエストをイシューで作りたい"
- "この問題をGitHubイシューとして記録して"

The agent will:
1. Determine issue type (bug, feature, improvement)
2. Gather relevant information
3. Create well-formatted issue
4. Include reproduction steps or requirements
5. Add labels and metadata
